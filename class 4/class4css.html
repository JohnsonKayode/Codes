CSS SYNTAX

p (selector) { (curly braces/ declaration block) Color: (Properties/declaration) red; (values)}

the whole thing is called a rule. The P is called a selector. It is followed by a set of declarations in a declaration block. Each declaration conntains properties and values. The selector, P in this case, tells the HTML what part of the HTML document should be styled by the declaration. The selector will style all the P elements on the page.

So basically, CSS is like a bunch of rules that the HTML has to abide by. So it dicates what rule the html should follow in terms of style and it properties.

Another rule of css is that it can be read from top to bottom.

however, what is above can be overwritten by what is below. So because it reads from top to bottom, The top takes effec but if there is another declaration below that has the same properties as the one above but different value, the one below takes precedence.

there are 4 main ways to use colors in CSS

  • Word: h1{color: red;}
  • Hex: h2{color:#ff0000;}
  • RGBa (red, green, blue, alpha(transparency) ): h5{color: rgba(255,0,0,1);}
  • HSLa(hue, saturation and lightness): span{color: hsla(0,100%, 50%, 1);}
